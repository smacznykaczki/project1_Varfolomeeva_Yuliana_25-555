Текстовая приключенческая игра "Лабиринт сокровищ"

Описание проекта: игра в консоли. Целью игры является поиск сокровища. Игрок перемещается между комнатами и решает загадки с целью найти сокровище.

Установка проекта в терминале:
# С помощью make
make install
# С помощью poetry
poetry install

Запуск проекта в терминале:
# С помощью make
make project
# С помощью poetry
poetry run project

Лицензия
