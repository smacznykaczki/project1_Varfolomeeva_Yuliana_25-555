# Лабиринт сокровищ

## 1. Описание проекта
Текстовая приключенческая игра на Python. Исследуйте таинственный лабиринт, собирайте предметы, решайте загадки и найдите спрятанное сокровище.

## 2. Установка проекта
Установите проект в терминале одним из способов:

_1. С помощью make_

    make install

_2. С помощью poetry_

    poetry install

## 3. Запуск проекта
Запустите проект в терминале одним из способов:

_1. С помощью make_

    make project

_2. С помощью poetry_

    poetry run project

## 5. Управление
Осуществляется процессом вода команд в терминале.


Основные команды:
+ **look** - Осмотреться в комнате
+ **go <direction>** - Пойти в направлении
+ **north/south/east/west** - Короткие команды перемещения
+ **take <item>** - Взять предмет
+ **use <item>** - Использовать предмет
+ **inventory** - Показать инвентарь
+ **solve** - Решить загадку
+ **help** - Показать справку
+ **quit** - Выйти из игры

## 6. Особенности реализации на Python
* Чистая архитектура - разделение ответственности между модулями.
* Словари и структуры данных - для хранения состояния игры.
* Функционально программирование - чистые функции для игровой логики.
* Обработка ввода - безопасный ввод пользователя с обработкой исключений.
* Псевдослучайность событий - реализация на основе математических функций.
